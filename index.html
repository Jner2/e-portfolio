Flood Monitoring System
Team Project  ·  My role: AI & Camera Integration

A real-time flood detection system using a custom-trained computer vision model. Two cameras monitor water levels using color-coded classification (Normal → Warning → Critical → Overflow). Camera 2 activates automatically when Camera 1 detects overflow — simulating a river-to-road flood scenario. Alerts are sent via a physical GSM module with live SIM card.
